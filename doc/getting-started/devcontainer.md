# Dev Container

ukoOS provides a dev container which contains all the tools you should need to get started. This is the easiest and recomended way to start working on ukoOS for most people.

## Using Visual Studio Code

Install [Visual Studio Code](https://code.visualstudio.com/Download) for your system and install the ["Dev Containers" extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers).
After cloning the repo and opening it in VS Code you should be prompted to "Reopen in dev container".
If not, you can open the command palette (Ctrl+Shift+P for Linux/Windows, Cmd+Shift+P for Mac) and type "Reopen in dev container" which should highlight the correct option (press enter to select it).
