# Z Standard Library Reference

This page attempts to document the entire public API of Z's standard library.

Everything on this page is available programmatically; insteaad of attempting to parse this page, use, create, or ask for the appropriate compiler API to extract this information.
A future version of this page should be generated by the compiler.

### `Z:T`

**Superclasses:** *None*

**Slots:** *None*

**Summary:**

All classes have `Z:T` as an effective superclass.
A variable of type `Z:T` can thus hold any value.

`Z:T` cannot be created with `Z:NEW`.

### `Z:ALLOCATE`

(*generic function*)

**Signature:** `(Z:ALLOCATE class &key &allow-other-keys) => object`

- `class`: [`Z:CLASS`](#zclass)
- `object`: `class`

**Summary:**

Allocates storage for an instance of a class, but does not initialize it.
This can be overridden to implement a variety of things:

- Singletons or object pools, in which calls to [`Z:NEW`](#znew) do not always result in allocating a new instance of the type, but instead may reuse an old one.

Users usually want [`Z:NEW`](#znew), not this function.
However, there are some cases where this function is useful:

- Allocating an instance of an object before using `Z:SLOT-VALUE` to fill in its slots, in order to intentionally skip calling `Z:INIT`.
