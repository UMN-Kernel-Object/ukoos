# Introduction

Welcome to ukoOS, a small operating system for RISC-V. The code can be found on [GitHub](https://github.com/UMN-Kernel-Object/ukoos).  


## UMN Kernel Object
The operating system is meant to be educational for the UMN Kernel Object club, so there are [tutorials](tutorials/tutorials.md) describing how to develop on it.


## Kernel documentation

In addition to tutorials, the kernel itself is [documented](kernel/kernel.md).
